# Phantom (Backend)

Application that allows simulating bus movements and enabling passengers to track their locations & movements. 
(Backend).

# Guidelines

## Clone the Repo

```
$ git clone https://github.com/atlp-rwanda/phatom-be-techode.git 
```

## Enter Directory

```
$ cd phatom-be-techode 
```

## install
```
$ npm i 
```

## Development

```
$ npm run dev
```

## Start Application

```
$ npm start
```
